# pixelRT
A simple 2d pixelated ray tracer.

## features
- completely vanilla java
- uses the dda algorithm <a src="https://en.wikipedia.org/wiki/Digital_differential_analyzer_(graphics_algorithm)">(https://en.wikipedia.org/wiki/Digital_differential_analyzer_(graphics_algorithm))</a>
- has very simple components (floors, walls, lights)
- accumalative frame rendering
- has very simple tonemapping
